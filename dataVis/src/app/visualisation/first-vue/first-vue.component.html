Bubble Map

A bubble chart is a variation of a scatter chart in which the data points are replaced with bubbles, and an additional dimension of the data is represented in the size of the bubbles. Just like a scatter chart, a bubble chart does not use a category axis â€” both horizontal and vertical axes are value axes. You can use a bubble chart if your data has three data series that each contain a set of values. The sizes of the bubbles are determined by the values in the third data series. Different bubble sizes are useful to visually emphasize specific values.
